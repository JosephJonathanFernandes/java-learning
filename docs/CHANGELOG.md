# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-28

### Added
- Modular architecture with proper Maven structure
- Student model with grade calculation logic
- Unit tests for Student model
- Example applications in examples package
- Security policy and environment configuration
- Comprehensive documentation

### Changed
- Refactored codebase to follow SOLID principles
- Improved project structure with src/, tests/, docs/, config/, scripts/
- Updated README and contributing guidelines
- Enhanced .gitignore for better security

### Removed
- Duplicate Student classes
- Loose Java files in root directory

### Fixed
- Package declarations and imports for all classes
- Test organization and coverage

## [0.1.0] - Initial Release

- Basic Java learning examples
- Simple student grade calculator
- Maven setup with JUnit
